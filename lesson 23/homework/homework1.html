თემა მოკლე აღწერა
HTML HTML არის markup ენა, რომელიც გამოიყენება ვებსაიტების სტრუქტურის შესაქმნელად.
CSS CSS უზრუნველყოფს HTML ელემენტების სტილიზაციას, როგორიცაა ფერები, ზომები და განლაგება.
JavaScript JavaScript გამოიყენება ვებსაიტებზე დინამიური ფუნქციების შესაქმნელად, მაგალითად, ვალიდაციისთვის და ანიმაციებისათვის.
Python Python არის სიმარტივით ცნობილი პროგრამირების ენა, რომელიც გამოიყენება ვებდეველოპმენტის, მონაცემთა ანალიზისა და აპლიკაციების შემუშავებისთვის.
